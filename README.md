# Opencv

 opencv work with python

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```bash
pip install opencv-python
```

## Usage

Run to "yüzsayma.py" file inside "opencv" folder.

```bash
python yüzsayma.py
```
